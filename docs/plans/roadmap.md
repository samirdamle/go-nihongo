# Roadmap — Go Nihongo!

## Now (v1)

Hybrid lookup web app: term dictionary cards + sentence translation/breakdown; local history/favorites; Vercel + Google Translation.

## Next (v1.x)

- PWA installability
- Labeled “AI assist” on hard misses (Gemini), clearly separated from dictionary
- Debounced live term lookup (dictionary only; still no live MT)
- Richer ranking / JLPT tags if data allows
- Copy buttons, share link with query params

## Later (v2+)

- Accounts + sync
- SRS from favorites
- TTS / speech input
- Offline term pack
- Native shell if web metrics demand it

## Not planned

- Replacing dictionary of record with an LLM
- Social/feed features
