# Vision — Go Nihongo!

## One-liner

**Go Nihongo!** helps English speakers look up and understand Japanese — whether they type romaji, Japanese script, or English — with clear Japanese forms, Hepburn readings, and meanings.

## Who it’s for

English-speaking Japanese learners who want a **fast, trustworthy study tool**: dictionary-quality term cards plus sentence help when they paste longer text.

## Job to be done

> When I encounter a Japanese word or phrase (or know it only in English/romaji), I want accurate Japanese form, pronunciation, and meaning — and for full sentences, a translation plus learnable word breakdown — so I can understand and remember without fighting the tool.

## Product principles

1. **Trust over fluency theater** — Readings and glosses ground in authoritative dictionary data; MT/LLM never silently invents dictionary cards.
2. **Hybrid by default, override when wrong** — Auto term vs sentence; user can flip.
3. **One clear input** — Mode tabs (romaji / Japanese / English) with auto-detect; not three competing textareas.
4. **Study-friendly cards** — Japanese · romaji (macrons) · meanings; furigana available on kanji.
5. **Deliberate lookup** — Explicit submit (no live API spam).
6. **Local memory first** — History and favorites without accounts in v1.

## Non-goals (v1)

- Accounts / cloud sync
- SRS, quizzes, courses
- Stroke order / handwriting
- TTS / voice input
- Full offline MT + offline dictionary
- Non-English app chrome
